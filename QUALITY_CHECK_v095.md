# QUALITY CHECK — v0.9.5 Thai Narrative Polish

## ตรวจ Flow

ผ่านลำดับหลักเดิมของเกม:

1. หน้าเริ่มเกม → ตั้งชื่อผู้นำและตระกูล
2. เข้าเกม → โหลด state ใหม่ด้วย save key v0.9.5
3. จัดแรงงาน
4. เลือกสิ่งก่อสร้าง / วิจัย
5. เลือกบทบาทผู้นำประจำเดือน
6. เลือกคำตอบเหตุการณ์
7. จบเดือน → คำนวณผลผลิต บริโภค ความเสี่ยง บาดเจ็บ ตาย ความทรงจำ และเหตุการณ์ต่อเนื่อง
8. แสดงบทสรุปรายเดือนด้วยภาษาไทยที่อ่านเป็นธรรมชาติขึ้น

## ตรวจภาษา

- แทนคำ UI อังกฤษที่ไม่จำเป็นด้วยคำไทย: พงศาวดาร, ความทรงจำ, เหตุการณ์ต่อเนื่อง, บันทึกเกม
- ปรับ Prologue ให้เป็นไทยเจ้าของภาษามากขึ้น
- ปรับ Monthly Resolution เป็น “บทสรุปของเดือน”
- ปรับการเสียชีวิตไม่ให้เป็นข้อความแห้ง แต่ยังคงความชัดเจนของสาเหตุ
- ปรับคำอธิบายระบบให้เข้ากับโทน survival / settlement / dynasty
