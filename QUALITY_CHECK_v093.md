# Quality Check — v0.9.3 Drama & Survival Loop

## ตรวจ Flow เกม

Flow หลักที่ออกแบบและตรวจในโค้ด:

1. หน้าเริ่มเกมตั้งชื่อผู้นำ + House + พื้นหลัง
2. เข้าเกมแล้วโหลด/สร้าง state เวอร์ชัน 0.9.3
3. หน้าเมืองแสดง KPI, ความเสี่ยง, เป้าหมายระยะ, แรงงาน และการกระทำผู้นำ
4. ผู้เล่นจัดแรงงานได้ไม่เกินแรงงานพร้อมใช้
5. เลือกสิ่งก่อสร้างได้เมื่อปลดล็อกและทรัพยากรพอ
6. เลือกวิจัยได้เมื่อ prerequisite ครบ
7. เลือกทางตอบเหตุการณ์ประจำเดือน
8. กดจบเดือนแล้วระบบคำนวณ:
   - ผลผลิต
   - การบริโภคอาหาร/น้ำ/ฟืน
   - อาหารเสีย
   - ความคืบหน้าก่อสร้าง
   - ความคืบหน้าวิจัย
   - ผลจากการกระทำผู้นำ
   - โรค/สัตว์ป่า/อุบัติเหตุ/ความขัดแย้ง/อากาศ
   - บาดเจ็บ/เสียชีวิต/รักษา
   - Event Chain และ Delayed Event
   - เปลี่ยนเดือน/ปี
   - เด็กโต, เด็กเกิด, ผู้เดินทางเข้าร่วม, ชราภาพ
   - Stage Objective และ Milestone
9. แสดง Monthly Resolution Modal
10. เลือกเหตุการณ์เดือนถัดไปจาก pending/delayed/weighted event pool

## ตรวจระบบ Realistic

มีระบบต่อไปนี้ในโค้ดแล้ว:

- Named villagers ไม่ใช่ตัวเลขลอย ๆ
- สุขภาพ, ความเหนื่อย, บาดเจ็บ, เสียชีวิต
- เสียชีวิตจากหลายสาเหตุแบบสมจริง
- ล่าสัตว์และก่อสร้างเสี่ยงอุบัติเหตุ
- ฤดูหนาวใช้ฟืนและทำให้หนาวตายได้
- น้ำไม่พอ/น้ำขุ่นทำให้สุขภาพลดและเกิดโรคได้
- อาหารเสียตามปริมาณ, ฤดูกาล, คลัง, การถนอมอาหาร
- โรคและแผลติดเชื้อแรงขึ้นเมื่อสุขภาพชุมชนต่ำหรือไม่มีคนดูแล
- ผู้เฒ่ามีโอกาสเสียชีวิตจากชราภาพเมื่อผ่านปี
- เด็กมีโอกาสโตเป็นแรงงานใหม่เมื่อครบปี
- คนใหม่/เด็กเกิดใหม่เกิดได้ตามเงื่อนไข
- Memory และ Chronicle เก็บผลของเหตุการณ์
- Event Chain เช่น คนแปลกหน้า, ข่าวลือ, เด็กเกิด, หลุมศพแรก
- Rare Events เช่น สุริยุปราคา, หินดำจากฟ้า, ฝูงกวางอพยพ
