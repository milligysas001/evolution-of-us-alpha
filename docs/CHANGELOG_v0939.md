# Evolution of Us v0.9.39 — Dynasty & Endgame

วันที่อัปเดต: 14 กรกฎาคม 2026

## 1. ครอบครัวและสายสัมพันธ์

- เพิ่มข้อมูลตระกูลให้บุคคล: ชื่อตระกูล พ่อแม่ คู่ครอง บุตร และบทบาทในครอบครัว
- เชื่อมสมาชิกเริ่มต้นของตระกูลผู้ก่อตั้งให้มีความสัมพันธ์จริง
- เด็กเกิดใหม่บันทึกพ่อแม่และเชื่อมกลับไปยังรายชื่อบุตรของผู้ปกครอง
- เมื่อมีทะเบียนครอบครัว ผู้ใหญ่ที่เหมาะสมสามารถสร้างครัวเรือนร่วมกันตามเวลา
- หน้าพงศาวดารแสดงสมาชิกตระกูลที่ยังมีชีวิตและสถานะของแต่ละคน

## 2. ทายาทและการสืบทอดผู้นำ

- เพิ่มรายชื่อผู้เหมาะสมเป็นทายาท โดยพิจารณาความสัมพันธ์ อายุ สุขภาพ ขวัญ และคุณลักษณะ
- ผู้เล่นแต่งตั้ง ยกเลิก หรือเปลี่ยนทายาทได้
- การแต่งตั้งล่วงหน้าช่วยลดผลกระทบด้านความไว้ใจและความเป็นปึกแผ่นเมื่อเปลี่ยนผู้นำ
- หากผู้นำเสียชีวิต ระบบเข้าสู่ขั้นตอนสืบทอดใน Monthly Pipeline
- เมื่อวิจัยการสืบทอดสายเลือด ผู้นำสูงวัยสามารถสละตำแหน่งได้
- หากไม่มีผู้พร้อมรับตำแหน่ง เมืองเข้าสู่สภาชั่วคราวและได้รับผลเสียจริง
- บันทึกชื่อผู้นำเดิม ผู้นำใหม่ ปี เดือน เหตุผล และรุ่นของตระกูล

## 3. เส้นทางชัยชนะ

เพิ่ม 6 เส้นทาง:

1. นครแห่งความอยู่รอด
2. ศูนย์กลางการค้า
3. สหพันธ์แห่งสันติ
4. นครแห่งความรู้
5. ตระกูลยืนยาว
6. อาณาจักรผู้พิทักษ์

แต่ละเส้นทางอ่านสถานะจริงจากอาหาร ประชากร ยุค การวิจัย อาคาร เมืองข้างเคียง สนธิสัญญา พันธมิตร ความมั่นคง และกองทัพ ผู้เล่นเลือกเส้นทางหลักได้ แต่ความคืบหน้าของเส้นทางอื่นยังคงอยู่

## 4. พงศาวดารตอนจบ

เมื่อเส้นทางที่เลือกสำเร็จ ระบบสร้างบทสรุปจากรอบเล่นจริง ได้แก่:

- ชื่อตระกูลและเส้นทางชัยชนะ
- ผู้นำรุ่นปัจจุบันและจำนวนรุ่น
- ระยะเวลาที่ใช้
- ประชากรและระดับเมือง
- เหตุการณ์สำคัญ การสูญเสีย และความสัมพันธ์กับเมืองอื่น
- ย่อหน้าสรุปมรดกที่ผู้เล่นทิ้งไว้

ผู้เล่นยังสามารถเล่นต่อเพื่อทำเส้นทางอื่นให้สำเร็จได้

## 5. UX/UI Audit ทุกหมวด

- เพิ่มกฎ `min-width: 0`, `overflow-wrap`, responsive grid และ horizontal scroll สำหรับตาราง/เมนูที่จำเป็น
- ปรับขนาดหัวข้อ เนื้อหา ป้ายสถานะ ปุ่ม ช่องกรอก และระยะห่างให้อ่านง่าย
- ลดพื้นที่ว่างโดยไม่ตัดข้อมูลสำคัญ
- ยกเลิกการตัดข้อความสำคัญด้วย ellipsis ในแถวแรงงาน ช่องบันทึก และข้อมูลหน้าหลัก
- ปรับแท็บให้ยืดหยุ่นตามความกว้างหน้าจอ และเลื่อนแนวนอนบนมือถือ
- ปรับหน้าพงศาวดาร เส้นทางชัยชนะ และตอนจบสำหรับ Desktop, Tablet และ Mobile
- รองรับ Dark Mode สำหรับองค์ประกอบใหม่

## 6. หน้าตั้งค่า

- นำการ์ด “ผู้มาตั้งถิ่นฐานประจำปี” ออกจากหน้าตั้งค่า
- คงระบบผู้มาตั้งถิ่นฐานภายในเกมไว้ เพราะยังเป็นส่วนของสมดุลประชากร
- คงช่องทางความคิดเห็นและตารางอันดับตระกูลไว้ตามเดิม
