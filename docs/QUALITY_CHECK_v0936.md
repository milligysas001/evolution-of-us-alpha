## Flow ที่ตรวจ

1. หน้าแรกอ่าน Autosave หลัง Hydration และแสดงช่องบันทึกด้วยตนเอง
2. Autosave เก็บสถานะครบและสำรองเซฟเดิมเมื่อเดือนเปลี่ยน
3. Manual Slot บันทึก โหลด ลบ และ Autosave ต่อในช่องที่กำลังใช้
4. Leader Board เก็บสถิติที่ดีที่สุดของแต่ละตระกูล
5. Event พ่อค้าเปิดแท็บพ่อค้าเร่เฉพาะช่วงมาเยือน
6. วิจัยการผลิตเหรียญ/ตลาดปิดแท็บพ่อค้าเร่และเปิดการค้าถาวร
7. Event พบเมืองเปิดแท็บเมืองข้างเคียง
8. สนธิสัญญาการค้าส่งผลทรัพยากรรายเดือน
9. ความตึงเครียดสูงสร้าง Event ชายแดน
10. ระบบทหารไม่แสดงในยุคแรก และเปิดเมื่อถึงยุคพร้อมวิจัย
11. กำลังพลใช้เสบียงจริงและถูกใช้ในผลสงคราม

## ขอบเขต

Leader Board รุ่นนี้เป็นอันดับบนเบราว์เซอร์/อุปกรณ์เดียวกัน เพราะโครงการยังไม่มีฐานข้อมูลผู้เล่นส่วนกลาง การทำอันดับออนไลน์ข้ามผู้เล่นต้องเชื่อม Backend เช่น Supabase ในเวอร์ชันถัดไป
