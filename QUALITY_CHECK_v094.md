# Quality Check — v0.9.4 Clarity, Villagers & Settlement Growth

## ตรวจ Flow เกม

Flow หลักที่ตรวจในโค้ด:

1. หน้าเริ่มเกมตั้งชื่อผู้นำ + House + พื้นหลัง
2. เข้าเกมแล้วโหลด/สร้าง state เวอร์ชัน 0.9.4
3. หน้าเมืองแสดง KPI, คุณภาพชีวิต, บัญชีทรัพยากร, คนสำคัญ, ความเสี่ยง, เป้าหมายระยะ, แรงงาน และการกระทำผู้นำ
4. ผู้เล่นจัดแรงงานเอง หรือกดแนะนำการจัดแรงงานตามสถานการณ์ได้
5. ระบบป้องกันไม่ให้จัดแรงงานเกินแรงงานพร้อมใช้
6. เลือกสิ่งก่อสร้างได้เมื่อปลดล็อกและทรัพยากรพอ
7. เลือกวิจัยได้เมื่อ prerequisite ครบ
8. เลือกทางตอบเหตุการณ์ประจำเดือน
9. กดจบเดือนแล้วระบบคำนวณ:
   - ผลผลิต
   - การบริโภคอาหาร/น้ำ/ฟืน
   - อาหารเสีย
   - ความคืบหน้าก่อสร้าง
   - ความคืบหน้าวิจัย
   - ผลจากการกระทำผู้นำ
   - โรค/สัตว์ป่า/อุบัติเหตุ/ความขัดแย้ง/อากาศ
   - บาดเจ็บ/เสียชีวิต/รักษา
   - Event Chain และ Delayed Event
   - เปลี่ยนเดือน/ปี
   - เด็กโต, เด็กเกิด, ผู้เดินทางเข้าร่วม, ชราภาพ
   - Stage Objective และ Milestone
10. แสดง Monthly Resolution Modal แบบแยกหมวด อ่านง่าย
11. เลือกเหตุการณ์เดือนถัดไปจาก pending/delayed/weighted event pool

- ตรวจว่า event IDs ที่ถูกเรียกจาก pending/delayed/fallback มีอยู่จริงทั้งหมด
- เพิ่ม Event Pool จาก 26 เป็น 41 เหตุการณ์
- เพิ่มระบบแนะนำแรงงานโดยอิงอาหาร, ผู้ป่วย, ฤดูกาล, งานก่อสร้าง, วิจัย และความเสี่ยงสัตว์ป่า
- เพิ่มเหตุผลของ Risk Forecast เพื่อลดความรู้สึกว่าสุ่มทำร้ายผู้เล่น
- เพิ่มผลกระทบของ delta.risk ให้เปลี่ยนค่าสุขภาพ ความไว้ใจ ความสามัคคี ความปลอดภัย และภัยภายนอกจริง
- เพิ่ม resource ledger ให้ผู้เล่นเห็นผลิต/ใช้/สุทธิ ก่อนจบเดือน
- เพิ่ม quality index: อาหาร น้ำ ที่พัก
- เพิ่ม settlement growth panel บอกขั้นเติบโตของถิ่นฐาน
- เพิ่ม monthly result categorization เพื่อตรวจผลลัพธ์ทีละหมวด

## ตรวจระบบ Realistic

มีระบบต่อไปนี้ในโค้ดแล้ว:

- Named villagers ไม่ใช่ตัวเลขลอย ๆ
- สุขภาพ, ความเหนื่อย, บาดเจ็บ, เสียชีวิต
- เสียชีวิตจากหลายสาเหตุแบบสมจริง
- ล่าสัตว์และก่อสร้างเสี่ยงอุบัติเหตุ
- ฤดูหนาวใช้ฟืนและทำให้หนาวตายได้
- น้ำไม่พอ/น้ำขุ่นทำให้สุขภาพลดและเกิดโรคได้
- อาหารเสียตามปริมาณ, ฤดูกาล, คลัง, การถนอมอาหาร
- โรคและแผลติดเชื้อแรงขึ้นเมื่อสุขภาพชุมชนต่ำหรือไม่มีคนดูแล
- ผู้เฒ่ามีโอกาสเสียชีวิตจากชราภาพเมื่อผ่านปี
- เด็กมีโอกาสโตเป็นแรงงานใหม่เมื่อครบปี
- คนใหม่/เด็กเกิดใหม่เกิดได้ตามเงื่อนไข
- Memory และ Chronicle เก็บผลของเหตุการณ์
- Event Chain เช่น คนแปลกหน้า, ข่าวลือ, ถ้ำเก่า, ฤดูหนาว, หลุมศพแรก
- Rare Events เช่น สุริยุปราคา, หินดำจากฟ้า, ฝูงกวางอพยพ
