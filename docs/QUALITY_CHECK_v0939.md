## สิ่งที่ตรวจเฉพาะ v0.9.39

- สร้างและปรับข้อมูลตระกูลจากบันทึกใหม่และบันทึกเก่า
- ผู้สมัครทายาทให้ความสำคัญกับญาติสายตรงที่มีสุขภาพพร้อม
- การเลือกเส้นทางชัยชนะไม่ลบความคืบหน้าของเส้นทางอื่น
- เส้นทางที่ครบเงื่อนไขสร้างพงศาวดารตอนจบ
- มีเส้นทางชัยชนะครบ 6 แบบและแสดงเหตุผลความคืบหน้า
- ระบบแก้ current leader ที่หายจากบันทึกเก่าได้
- Monthly Pipeline มีขั้นตอนสืบทอดผู้นำ
- หน้าตั้งค่าไม่มีการ์ดผู้มาตั้งถิ่นฐานประจำปี
- CSS ใหม่มีข้อกำหนดป้องกัน overflow และรองรับ responsive

## ข้อจำกัดการตรวจภาพ

ตรวจโครงสร้าง Responsive CSS, การ Build และการเปิดเส้นทางเว็บจริงแล้ว แต่สภาพแวดล้อมทดสอบไม่สามารถสร้างภาพหน้าจอ Headless Chromium จนจบได้ จึงควรตรวจรอบสุดท้ายบน Chrome จริงที่ความกว้างประมาณ 1440, 1024, 768 และ 390 พิกเซลหลังติดตั้งบนเครื่องผู้ใช้
