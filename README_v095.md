# Evolution of Us v0.9.5 — Thai Narrative Polish

เวอร์ชันนี้ต่อยอดจาก v0.9.4 โดยเน้นให้ภาษาไทยในเกมอ่านเป็นธรรมชาติขึ้น เหมือนงานเขียนเกมภาษาไทยที่ตั้งใจทำจริง ไม่ใช่ข้อความแปลตรงตัว

## จุดที่อัปเดต

- ปรับหน้าเริ่มเกม Prologue ให้เป็นภาษาไทยเชิงเล่าเรื่อง อ่านลื่นและอินขึ้น
- ปรับคำเรียกใน UI ให้สอดคล้องกัน เช่น พงศาวดาร, ความทรงจำ, เหตุการณ์ต่อเนื่อง, บันทึกเกม
- ปรับบทสรุปรายเดือนให้มีน้ำหนักแบบนิยาย survival มากขึ้น
- ปรับข้อความการตาย การเกิด การเข้าร่วมของผู้เดินทาง และเหตุการณ์สำคัญให้อ่านเป็นธรรมชาติขึ้น
- ลดคำอังกฤษที่ไม่จำเป็นในหน้าเกม เช่น Cost, Work, Prereq, Save, Chronicle, Memory
- ยังรักษา flow เกมเดิมของ v0.9.4 ไว้ครบ: จัดแรงงาน → สร้าง/วิจัย → เลือกผู้นำ → ตอบเหตุการณ์ → จบเดือน

## วิธีติดตั้ง

แตก ZIP แล้วรันใน PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v095-thai-narrative.ps1
```

จากนั้นเปิด:

```text
http://localhost:3000
```

แนะนำให้เริ่มเซฟใหม่ เพราะ save key เปลี่ยนเป็น `eou-v095-save` เพื่อแยกจากเวอร์ชันเก่า
