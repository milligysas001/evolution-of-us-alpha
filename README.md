# Evolution of Us v0.9.19 — Compact Labor & Quiet Event Flow

ปรับ flow ให้เหมาะกับการเล่นบนเว็บมากขึ้น:
- Event ปกติ/การกระทำผู้นำไม่เด้งกลางจออัตโนมัติแล้ว
- กระดิ่งแจ้งเฉพาะเหตุการณ์สำคัญ เช่น พ่อค้า โจร ขโมยเสบียง ผู้อพยพ การเกิด หรือเหตุการณ์หายาก
- หน้าแรงงานรายบุคคลเปลี่ยนเป็นแถวแนวยาว ประหยัดพื้นที่สำหรับประชากรจำนวนมาก
- เพิ่ม filter คนตามความถนัด/บทบาท เช่น พราน ช่าง ผู้รักษา ผู้จดจำ เวรยาม เด็กช่วยงาน ผู้สูงอายุ
- แก้ logic “แรงงานเกิน” ให้ดูจากจำนวนคนที่ถูกใช้จริง ไม่ใช่ผลผลิตหลังบัฟความถนัด
- เด็ก 12–14 และผู้สูงอายุ 60+ ยังทำงานได้แบบประสิทธิภาพลดลง หากมี passive เหมาะสมจะช่วยได้มากขึ้น
