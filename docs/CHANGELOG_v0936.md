# Evolution of Us v0.9.36 — Save, Leader Board, Neighbor Cities & Military

## ระบบบันทึก

- เพิ่ม Autosave ที่เก็บสถานะเกมครบทุกระบบ
- เก็บสำเนาก่อนข้ามเดือน เพื่อกู้กลับเมื่อเซฟล่าสุดมีปัญหา
- เพิ่มบันทึกด้วยตนเอง 3 ช่อง ตั้งชื่อ โหลด ลบ และบันทึกทับได้
- ช่องที่กำลังเล่นจะถูก Autosave ต่อเนื่องโดยอัตโนมัติ
- เพิ่มดาวน์โหลดไฟล์เซฟ JSON และระบบนำเข้าเดิม
- หน้าเริ่มเกมตรวจเซฟหลัง Hydration ถูกต้อง และโหลดบันทึก 3 ช่องได้โดยตรง

## Leader Board

- เพิ่ม Leader Board ในหน้าตั้งค่า
- แสดงอันดับ ชื่อตระกูล ผู้นำ ขนาดเมือง เวลาที่ทำได้ และจำนวนประชากร
- จัดลำดับตามยุคที่ไปถึง เวลารอด ประชากร และค่าสภาพเมือง
- เก็บคะแนนที่ดีที่สุดของแต่ละตระกูลบนอุปกรณ์เดียวกัน

## ข่าว พ่อค้า และการค้า

- แยกแท็บข่าวสารออกจากหน้าพ่อค้าและตลาด
- แท็บพ่อค้าเร่แสดงเฉพาะเดือนที่พ่อค้า/คาราวานมาถึง
- เมื่อวิจัยระบบเงิน/ตลาด หรือสร้างโครงสร้างการค้าสำเร็จ แท็บพ่อค้าเร่จะถูกแทนด้วยแท็บการค้าถาวร
- การซื้อขายตรวจช่องทางค้าและทรัพยากรจริง

## เมืองข้างเคียง

- เพิ่ม Event พบควันไฟของเมืองอื่น และ Event ข้อพิพาทชายแดน
- เพิ่มแท็บเมืองข้างเคียงหลังค้นพบเมืองแรก
- แต่ละเมืองมีประชากร ขนาดเขตแดน ความสัมพันธ์ ความไว้ใจ ความกลัว ความตึงเครียด ความชำนาญ และสถานะสงคราม
- ทำกิจกรรมได้จริง: ส่งทูต ของขวัญ สนธิสัญญาการค้า แลกเปลี่ยน อ้างสิทธิ์ เปิดศึก ขอสงบศึก และพันธมิตร
- สนธิสัญญาการค้าส่งทรัพยากรทุกเดือน สงครามใช้กำลังและทำให้เสียเสบียง/บาดเจ็บได้

## การทหาร

- ระบบทหารถูกซ่อนในยุคแรก
- เปิดหลังเข้าสู่หมู่บ้านถาวรและวิจัยระบบกองอาสาป้องกันเมือง
- เพิ่มงานวิจัยกองอาสา กองกำลังประจำเมือง และเสบียงทางทหาร
- เพิ่มลานฝึกกองอาสาและค่ายทหารประจำเมือง
- กำลังพลมีค่าเลี้ยงดู ความพร้อม ขวัญ อุปกรณ์ ประสบการณ์ และท่าทีจริง
- พลังรบถูกใช้คำนวณการปะทะกับเมืองข้างเคียง

## UI และ Flow

- ย่อแถวจัดแรงงานให้เป็นแถบแนวนอนสั้นลง โดยคงชื่อ อายุ สถานะ คุณลักษณะ งาน การพัก สุขภาพ ความล้า และช่องเลือกงาน
- ตรวจเส้นทาง Save → Home → Load, Merchant Event → Merchant Tab → Trade Research → Permanent Trade Tab, Neighbor Event → Neighbor Tab → Diplomacy/War และ Military Research → Building → Upkeep → Border Combat
