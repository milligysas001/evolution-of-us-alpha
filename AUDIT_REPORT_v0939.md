# Evolution of Us v0.9.39 — รายงานตรวจระบบ Dynasty & Endgame

วันที่ตรวจ: 14 กรกฎาคม 2026

## ขอบเขต

เวอร์ชันนี้เพิ่มวงจรการเล่นระยะยาวให้เกมมี “คนรุ่นต่อไป” และ “เป้าหมายปลายทาง” โดยเชื่อมครอบครัว ทายาท การสืบทอด เส้นทางชัยชนะ และบทสรุปเข้ากับ Monthly Pipeline, Save Migration, Seeded Random, เมืองข้างเคียง การค้า การวิจัย และการทหารจากเวอร์ชันก่อนหน้า

## 1. ระบบครอบครัว

บุคคลรองรับข้อมูล `houseName`, `parentIds`, `spouseId`, `childrenIds` และ `familyRole` สมาชิกตระกูลเริ่มต้นถูกเชื่อมความสัมพันธ์ตั้งแต่สร้างเกม เด็กเกิดใหม่ผูกกับพ่อแม่และข้อมูลบุตรของผู้ปกครอง ไม่ได้เป็นเพียงจำนวนประชากรที่เพิ่มขึ้น

เมื่อมีงานวิจัยบันทึกสายสัมพันธ์ ผู้ใหญ่ที่มีสุขภาพเหมาะสมและยังไม่มีคู่สามารถสร้างครัวเรือนใหม่ การเปลี่ยนแปลงถูกบันทึกในพงศาวดารและใช้ต่อในระบบทายาท

## 2. ทายาทและการสืบทอด

ผู้สมัครทายาทถูกจัดคะแนนจาก:

- ความสัมพันธ์ทางสายเลือดกับผู้นำ
- อายุที่พร้อมรับหน้าที่
- สุขภาพและขวัญกำลังใจ
- คุณลักษณะและความพร้อมในการเป็นผู้นำ
- การแต่งตั้งล่วงหน้าโดยผู้เล่น

การสืบทอดเกิดใน Monthly Pipeline เมื่อผู้นำเสียชีวิต หรือเมื่อผู้นำสูงวัยและปลดล็อกการสืบทอดสายเลือด ระบบเปลี่ยนชื่อผู้นำ อายุ บทบาท รุ่นของตระกูล ความไว้ใจ ความเป็นปึกแผ่น พงศาวดาร และการแจ้งเตือนจริง

กรณีไม่มีผู้สมัครที่พร้อม ระบบเข้าสู่สภาชั่วคราวและลดความไว้ใจ ความปลอดภัย และความเป็นปึกแผ่น โดยยังเก็บ ID ผู้นำเดิมไว้เพื่อให้ Save Schema ไม่เสียหาย

## 3. เส้นทางชัยชนะ

มี 6 เส้นทางที่คำนวณจากสถานะจริง:

| เส้นทาง | ระบบหลักที่ใช้ตรวจ |
|---|---|
| นครแห่งความอยู่รอด | เสบียง วิกฤต ประชากร สุขภาพ และยุค |
| ศูนย์กลางการค้า | ตลาด คาราวาน ทอง สนธิสัญญา และเมืองการค้า |
| สหพันธ์แห่งสันติ | พันธมิตร ความสัมพันธ์ ความไว้ใจ และการไม่มีสงคราม |
| นครแห่งความรู้ | จำนวนงานวิจัย ความรู้ พงศาวดาร และยุค |
| ตระกูลยืนยาว | จำนวนรุ่น ประชากร ความเป็นปึกแผ่น และอายุเมือง |
| อาณาจักรผู้พิทักษ์ | ยุค ความปลอดภัย กำลังทหาร ความพร้อม และป้อมปราการ |

ผู้เล่นเลือกเส้นทางหลักได้จากหน้าพงศาวดาร การประเมินเกิดหลังจบเดือน เมื่อครบเงื่อนไข ระบบเพิ่มชัยชนะ แจ้งเตือน บันทึกเวลา และสร้างตอนจบโดยไม่บังคับหยุดเล่น

## 4. พงศาวดารตอนจบ

บทสรุปใช้ข้อมูลจากรอบเล่นจริง ได้แก่ ชื่อตระกูล ผู้นำรุ่นล่าสุด จำนวนรุ่น ปีและเดือน จำนวนประชากร ระดับเมือง เมืองพันธมิตร การสูญเสีย และเหตุการณ์สำคัญ จึงทำให้แต่ละรอบมีตอนจบที่ต่างกันตามการตัดสินใจ

## 5. Save Migration และ Validation

- Game Version: 0.9.39
- Schema Version: 5
- บันทึกเก่าเติมข้อมูลครอบครัวให้ทุกคน
- สร้าง `dynasty` และ `victory` เมื่อไม่มี
- ซ่อม current leader และ designated heir ที่อ้างถึงบุคคลซึ่งไม่มีชีวิตแล้ว
- ตรวจ array ของพ่อแม่ บุตร ประวัติการสืบทอด และชัยชนะ
- Checksum, Seed/RNG State, Autosave, Backup และ Manual Slots ยังทำงานต่อเนื่อง

## 6. UX/UI ทุกหมวด

เพิ่มกฎกลางเพื่อป้องกันข้อความล้นและซ้อนทับ:

- `min-width: 0` สำหรับ grid/flex children ที่มีข้อมูลยาว
- `overflow-wrap: anywhere` สำหรับข้อความผู้ใช้ ชื่อ และคำอธิบาย
- แท็บแบบ auto-fit และเลื่อนแนวนอนบนหน้าจอเล็ก
- ตารางรายงานและ Leader Board เลื่อนแนวนอนได้
- หัวข้อใช้ `clamp()` ให้พอดีกับหน้าจอ
- ยกเลิก ellipsis ในข้อมูลสำคัญของแรงงานและช่องบันทึก
- ป้ายสถานะและปุ่มยอมขึ้นบรรทัดใหม่แทนการซ้อน
- หน้าตระกูลและชัยชนะลดจาก 3 คอลัมน์เป็น 2 และ 1 ตามขนาดจอ
- Dark Mode ครอบคลุมการ์ดใหม่

หน้าตั้งค่าถูกตัดการ์ดผู้มาตั้งถิ่นฐานประจำปีออกแล้วตามข้อกำหนด โดยไม่เปลี่ยนกลไกประชากรภายในเกม
