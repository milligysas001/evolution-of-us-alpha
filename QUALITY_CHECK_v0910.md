# Evolution of Us v0.9.10 — Adaptive Layout + Split Project Tabs

อัปเดตนี้จัดระเบียบ UI/Flow ให้เหมาะกับการเล่น Alpha มากขึ้น

## เพิ่ม/ปรับหลัก
- รวมการเลือก “การกระทำของผู้นำ” และ “เหตุการณ์ประจำเดือน” ไว้ในกรอบตัดสินใจเดียวกัน
- ต้องเลือกทั้งสองอย่างก่อนจบเดือน ถ้าไม่เลือกจะจบเดือนไม่ได้
- แสดงทองเป็น “คลังเมือง” บนแถบด้านบน
- ซ่อนข้อมูลที่รกเกินไป เช่นรายการงานที่ยังไม่ปลดล็อก และย่อ Debug Report ไว้เฉพาะใน Settings
- คนในค่ายสุ่มใหม่ทุกครั้งเมื่อเริ่มเกมใหม่ ทำให้แต่ละรอบมีผู้เล่น/ชาวบ้านชุดใหม่
- ก่อสร้างและวิจัยที่กำลังดำเนินการแสดงบนหน้าหลัก พร้อมความคืบหน้าและคาดการณ์เดือนที่เหลือ
- รวมแท็บก่อสร้าง/วิจัยเป็น “โครงการ” เพื่อให้หมวดหมู่กระชับขึ้น
- เหตุการณ์พิเศษ เช่น พ่อค้ามาถึง ภัยมนุษย์ เหตุการณ์หายาก จะแสดงเป็นกรอบพิเศษให้ตัดสินใจ
- พ่อค้าสามารถซื้อ/ขายทรัพยากรบางอย่างเพื่อเพิ่มทองได้

## Flow ที่ตรวจ
หน้าแรก → เริ่มเกม → สุ่มคนในค่าย → จัดแรงงาน → เลือกโครงการ → เลือกผู้นำ + เหตุการณ์ → จบเดือน → สรุปผล → เดือนถัดไป


## v0.9.10 Update
- แยกแท็บก่อสร้างและวิจัยออกจากกันเพื่อให้จัดการง่ายขึ้น
- เพิ่มระบบตรวจจับ Desktop / Tablet / Mobile และปรับ layout อัตโนมัติ
- ปรับคำแนะนำและข้อความ tutorial ให้เป็นภาษาไทยร่วมสมัย อ่านง่าย และสากลขึ้น
- ปรับหน้าโครงการให้แสดงระยะเวลาคงเหลือของก่อสร้าง/วิจัยชัดเจนขึ้น
