# Evolution of Us v0.9.41 — Wide Layout & Living Chronicle

## UX/UI

- ยกเลิกแผงตัดสินใจด้านขวา และขยายเนื้อหาหลักให้ใช้ความกว้างที่เหลือทั้งหมด
- เพิ่มแท็บ `✦ ตัดสินใจเดือนนี้` ในเมนูหลัก พร้อมจุดเตือนเมื่อยังเลือกไม่ครบ
- จัดหน้าตัดสินใจเป็น 2 ส่วนบนจอกว้าง และเรียงเป็นแนวตั้งบนจอขนาดเล็ก
- เพิ่มขนาดข้อความสำคัญ ระยะบรรทัด และพื้นที่กด โดยไม่ทำให้การ์ดฟุ่มเฟือย
- ซ่อนเมนูด้านล่างบนเดสก์ท็อปเพื่อลดเมนูซ้ำ และคงไว้สำหรับแท็บเล็ต/มือถือ

## พงศาวดาร

- บันทึกหนึ่งบทสรุปหลักต่อเดือน
- รวมความคืบหน้าก่อสร้าง วิจัย ทรัพยากร และเหตุการณ์ไว้ในบทสรุปเดียว
- บันทึกแยกเฉพาะการเปลี่ยนยุค การสืบทอด ชัยชนะ สงคราม เมืองข้างเคียง การตั้งชื่อ และผู้เสียชีวิต
- กรองรายการย่อยจากบันทึกเก่าเมื่อโหลดเกม

## ครอบครัวและทายาท

- ตรวจเงื่อนไขเปิดระบบทายาทจากทะเบียนครอบครัวหรือยุคหมู่บ้านถาวร
- แสดงทายาทปัจจุบันหลังคลิกทันที
- แสดงข้อความยืนยันในสถานะบันทึก พร้อมปุ่มยกเลิก
- เปลี่ยนบทบาททายาทเก่ากลับเป็นสมาชิกตระกูลเมื่อแต่งตั้งคนใหม่

## ชื่อบุคคลและชื่อถิ่นฐาน

- เพิ่มชื่อจากแนวกรีก อียิปต์ เมโสโปเตเมีย เปอร์เซีย เคลต์ นอร์ดิก อินเดีย แอฟริกา อาหรับ สลาฟ เอเชียตะวันออก และชื่อแนวชายแดน
- สุ่มชื่อโดยไม่ซ้ำภายในรายชื่อที่มีชีวิตและผู้มาใหม่
- เพิ่มชื่อถิ่นฐานใน Game State, Save, Top Bar และ Chronicle
- เปิดหน้าตั้งชื่อใหม่เมื่อพัฒนาเป็นชุมชน หมู่บ้าน เมือง นครรัฐ และอาณาจักร

## เส้นทางชัยชนะ

- ยกเลิกการเลือกเส้นทางชัยชนะล่วงหน้า
- แสดงทั้ง 6 เส้นทางเป็นการ์ดข้อมูลเท่านั้น
- ตรวจทุกเส้นทางอัตโนมัติเมื่อจบเดือน
- ปรับเกณฑ์ตามระดับ เน้นเรื่องราว / สมดุล / เอาชีวิตรอด / ไอรอนแมน

## บันทึกเกม
