# Evolution of Us v0.9.3 — Drama & Survival Loop

เวอร์ชันนี้เป็นแนวเริ่มจากศูนย์: ตระกูลของผู้เล่น + ชาวบ้าน 10 คน + ค่ายพักแรมที่ยังไม่มีเมือง ไม่มีบ้านถาวร และไม่มีเนื้อเรื่องบังคับล่วงหน้า

## ระบบหลัก

- เล่นเป็นเทิร์นรายเดือน
- จัดแรงงาน: หาอาหาร, ตัดไม้, เก็บหิน, ก่อสร้าง, เฝ้ายาม, ดูแลคนป่วย, ทดลอง/เรียนรู้
- เลือกก่อสร้างและวิจัย
- เลือกการกระทำของผู้นำ
- ตอบเหตุการณ์ประจำเดือน
- จบเดือนแล้วระบบคำนวณผลผลิต การบริโภค ความเสี่ยง อุบัติเหตุ โรค ความตาย การเกิด Memory และ Event Chain

## ระบบ Realistic ที่เพิ่ม

- ชาวบ้านมีชื่อ อายุ สุขภาพ กำลังใจ ความเหนื่อย บาดเจ็บ และเสียชีวิตได้จริง
- สาเหตุเสียชีวิต: อดอาหาร, หนาวจัด, โรค, แผลติดเชื้อ, อุบัติเหตุล่าสัตว์, งานก่อสร้าง, สัตว์ป่า, ชราภาพ
- ฤดูกาลมีผลต่ออาหาร น้ำ ฟืน โรค และสภาพอากาศ
- อาหารเสียถ้าไม่มีคลัง/ถนอมอาหาร
- น้ำและฟืนเป็นทรัพยากรที่ต้องบริหาร
- งานหนักเกินไปเพิ่มความเหนื่อยและอุบัติเหตุ
- คนป่วย/บาดเจ็บรักษาได้ด้วยแรงงานดูแล สมุนไพร และกระท่อมหมอยา
- มีระบบข่าวลือ, Event Chain, Memory, Milestone, ผู้จากไป, เป้าหมายรายระยะ และเส้นทางพัฒนา

## วิธีติดตั้ง

แตก ZIP แล้วรันใน PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v093-drama-survival.ps1
```

ค่าเริ่มต้นจะติดตั้งไปที่:

```text
C:\Users\phass\evolution-of-us
```

ถ้าโปรเจกต์อยู่คนละที่:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v093-drama-survival.ps1 -ProjectPath "D:\your-project"
```

หลังขึ้น Ready ให้เปิด:

```text
http://localhost:3000
```
