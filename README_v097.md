# Evolution of Us v0.9.7 — Guided Start & Dynamic Leader Actions

เวอร์ชันนี้ต่อยอดจาก v0.9.6 โดยเพิ่มระบบสอนเล่นตอนเริ่มเกม และปรับ “การกระทำของผู้นำเดือนนี้” ให้เปลี่ยนตามสถานการณ์จริงของเกม

## เพิ่มใหม่

- Tutorial เด้งขึ้นหลังเริ่มเกมครั้งแรก
- มีปุ่ม ถัดไป / ย้อนกลับ / ข้าม / ไม่แสดงอีก
- ตั้งค่าเปิด Tutorial อีกครั้งได้
- การกระทำของผู้นำเป็น Dynamic ไม่ใช่ชุดเดิมตลอด
- ตัวเลือกผู้นำอิงจาก:
  - ความเสี่ยงอาหาร
  - โรคและผู้บาดเจ็บ
  - สัตว์ป่า / ภัยภายนอก
  - ความขัดแย้ง / ความไว้ใจ / ความยุติธรรม
  - ฤดูหนาว / ฤดูฝน / ฟืน / เครื่องมือ
  - ผู้เสียชีวิตล่าสุด
  - เหตุการณ์ประจำเดือนที่กำลังเกิดขึ้น
- ตัวเลือกบางอย่างถูกล็อกตามเงื่อนไข เช่น ต้องมีสมุนไพร ต้องมีฟืน ต้องมีไม้
- เพิ่มผลจริงของตัวเลือกใหม่ เช่น ตรวจคลัง นำคนหาอาหาร ต้มสมุนไพร เดินเวรยาม ตามรอยสัตว์ ตั้งกติกา แก้รอยรั่ว ซ่อมเครื่องมือ พิธีรำลึกผู้จากไป
- รีเซ็ตการกระทำผู้นำเป็นค่าเริ่มต้นทุกเดือน เพื่อให้ผู้เล่นเลือกใหม่ตามสถานการณ์
- เปลี่ยน save key เป็น `eou-v097-save`

## วิธีติดตั้ง

แตก ZIP แล้วเปิด Terminal ในโฟลเดอร์ที่มีไฟล์นี้ จากนั้นรัน:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v097-guided-survival.ps1
```

เปิดเกม:

```text
http://localhost:3000
```

แนะนำให้เริ่มเกมใหม่ เพราะระบบ Tutorial และ Dynamic Leader Actions ใช้ save key ชุดใหม่
