# Evolution of Us v0.9.41 — Audit Report

## Release: Wide Layout & Living Chronicle

วันที่ตรวจ: 14 กรกฎาคม 2026

## ขอบเขตการอัปเดต

### 1. หน้าหลักและการตัดสินใจ

- นำแผงอ่าน/ตัดสินใจด้านขวาออกจากโครงสร้างหน้าเกมทั้งหมด
- เปลี่ยนหน้าหลักเป็น 2 คอลัมน์: แถบข้อมูลเมือง + เนื้อหาหลัก
- เพิ่มแท็บ `✦ ตัดสินใจเดือนนี้` ในแถบเมนูด้านบน
- แสดงจุดเตือนเมื่อยังไม่ได้เลือกการนำของผู้นำหรือคำตอบเหตุการณ์
- ปุ่มเมนูย่อบนจอเล็กจะพาไปแท็บตัดสินใจ ไม่จบเดือนข้ามเนื้อหา
- หน้าตัดสินใจใช้ 2 คอลัมน์บนจอกว้าง และ 1 คอลัมน์บนแท็บเล็ต/มือถือ
- ปรับขนาดตัวอักษร ระยะบรรทัด ระยะกด และการตัดบรรทัดให้สบายตาขึ้น

### 2. พงศาวดารแบบกระชับ

- ใช้หนึ่งบทสรุปหลักต่อเดือนเป็นแกนของพงศาวดาร
- รายงานเหตุการณ์ ทางเลือก ประชากร อาหาร น้ำ ความปลอดภัย และจุดสำคัญในรายการเดียว
- ไม่เก็บรายการก่อสร้าง วิจัย ผลผลิต และการกระทำย่อยเป็นบรรทัดแยกทุกครั้ง
- เก็บแยกเฉพาะการเปลี่ยนยุค การตั้งชื่อ การสืบทอด ชัยชนะ สงคราม เมืองข้างเคียง และผู้เสียชีวิต
- เมื่อโหลดบันทึกเก่า ระบบกรองรายการที่ไม่ใช่เหตุการณ์หลักออกจากหน้าอ่านพงศาวดาร

### 3. ระบบทายาท

- ตรวจเงื่อนไขเปิดระบบจาก `ทะเบียนครอบครัว` หรือยุค `หมู่บ้านถาวร`
- คลิกการ์ดผู้สมัครหรือเลือกจากรายการแล้วสถานะเปลี่ยนทันที
- แสดงแถบ `ทายาทปัจจุบัน` พร้อมคำอธิบายและปุ่มยกเลิก
- แสดงข้อความยืนยันในสถานะบันทึกเกม
- เมื่อเปลี่ยนทายาท บทบาทของทายาทคนเก่าจะถูกคืนค่าอย่างเหมาะสม
- การสืบทอดยังใช้ผู้ที่แต่งตั้งเป็นลำดับแรกเมื่อผู้นำเสียชีวิตหรือสละตำแหน่ง

### 4. ระบบชื่อบุคคล

เพิ่มคลังชื่อจากหลายแนววัฒนธรรม ได้แก่:

- กรีก
- อียิปต์
- เมโสโปเตเมีย
- เปอร์เซีย
- เคลต์
- นอร์ดิก
- อินเดีย
- แอฟริกา
- อาหรับ
- สลาฟ
- เอเชียตะวันออก
- ชื่อแนวผู้ตั้งถิ่นฐาน

การสุ่มชื่อ:

- ไม่ซ้ำภายในรายชื่อเริ่มต้น
- ผู้มาใหม่และเด็กเกิดใหม่ตรวจรายชื่อที่มีอยู่ก่อนตั้งชื่อ
- รายชื่อผู้ย้ายถิ่นใน Event ใช้การเลือกแบบ deterministic เพื่อไม่ให้การเปิดหน้าจอเปลี่ยนสถานะ RNG
- หากใช้ชื่อหมดจึงเติมหมายเลขท้ายอย่างเป็นระบบ

ผลจำลอง 250 เกม:

- ประชากรเริ่มต้น: 15 คนทุกครั้ง
- ชื่อซ้ำภายในชุดเริ่มต้น: 0 ครั้ง
- ชื่อที่สุ่มพบแตกต่างกัน: 176 ชื่อ

### 5. การตั้งชื่อถิ่นฐาน

เพิ่มข้อมูลใน Game State และ Save:

- `settlementName`
- `pendingSettlementRename`
- `lastNamedStage`
- `settlementNameHistory`

Flow:

1. เริ่มต้นด้วยชื่อค่ายตามตระกูล
2. เมื่อเข้าสู่ชุมชนแรกเริ่ม ให้ตั้งชื่อชุมชน
3. เมื่อเป็นหมู่บ้านถาวร ให้ตั้งชื่อหมู่บ้าน
4. เมื่อเข้าสู่ช่วงเมือง ให้ตั้งชื่อเมืองใหม่
5. เมื่อเป็นนครรัฐและอาณาจักร ให้ตั้งชื่อใหม่อีกครั้ง

ชื่อจะแสดงใน Top Bar, Save, รายงาน, พงศาวดาร และบทสรุปตอนจบ

### 6. เส้นทางชัยชนะ

- ยกเลิกการเลือกเส้นทางล่วงหน้าจากหน้า UI
- การ์ดชัยชนะเป็นข้อมูลสำหรับอ่านและติดตามเท่านั้น
- ระบบตรวจทั้ง 6 เส้นทางอัตโนมัติหลังจบเดือน
- สามารถบรรลุหลายเส้นทางในรอบเล่นเดียว
- เกณฑ์เปลี่ยนตามระดับความยาก:
  - เน้นเรื่องราว: ลดเกณฑ์
  - สมดุล: เกณฑ์มาตรฐาน
  - เอาชีวิตรอด: เพิ่มเกณฑ์
  - ไอรอนแมน: เพิ่มเกณฑ์สูงสุด
- บันทึกเก่าที่เคยเลือกเส้นทางจะถูก Migration ให้ `chosenPath = null`
