# Evolution of Us — Game Design Spec (v0.9.15)

## Core Loop
ตั้งถิ่นฐานจากศูนย์ → จัดแรงงานรายเดือน → เลือกก่อสร้าง/วิจัย → เลือกการกระทำผู้นำ → ตอบเหตุการณ์ → จบเดือน → บันทึกพงศาวดาร

## Design Pillars
1. ทุกชีวิตมีค่า
2. ทุกเดือนต้องมีความเสี่ยงและทางเลือก
3. เมืองเติบโตจากการตัดสินใจ ไม่ใช่จากตัวเลขลอย ๆ
4. ข้อมูลเกมต้อง portable เพื่อย้ายไป Godot ได้

## Water Design
น้ำเป็นทรัพยากรเอาตัวรอดพื้นฐาน จึงต้องหาได้ตั้งแต่เดือนแรกผ่านงาน “ตักน้ำ / ดูแลน้ำสะอาด” แต่คุณภาพและปริมาณจะดีขึ้นเมื่อวิจัยการหาแหล่งน้ำ สร้างบ่อน้ำ และพัฒนาสุขาภิบาล
