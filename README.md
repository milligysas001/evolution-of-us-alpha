# Evolution of Us — รุ่นทดสอบ v0.9.39 Dynasty & Endgame

เวอร์ชันนี้ต่อยอดจาก v0.9.38 โดยทำให้ “ตระกูล” เป็นระบบที่เล่นต่อเนื่องได้จริง ตั้งแต่ครอบครัว การแต่งตั้งทายาท การเปลี่ยนผู้นำหลายรุ่น ไปจนถึงเส้นทางชัยชนะและพงศาวดารตอนจบ พร้อมตรวจ UX/UI ทุกหมวดให้กระชับ อ่านง่าย และลดความเสี่ยงข้อความซ้อนทับกัน

## จุดสำคัญ

- สมาชิกมีข้อมูลพ่อแม่ คู่ครอง บุตร และบทบาทในตระกูล
- ครัวเรือนใหม่เกิดขึ้นตามเวลาและเงื่อนไขของเมือง
- แต่งตั้งหรือยกเลิกทายาทได้จากหน้าพงศาวดาร
- ผู้นำเสียชีวิตหรือสละตำแหน่งแล้วระบบเลือกผู้สืบทอดจริง
- บันทึกผู้นำแต่ละรุ่น เหตุผลการสืบทอด และเหตุการณ์สำคัญของตระกูล
- เส้นทางชัยชนะ 6 แบบ พร้อมความคืบหน้าที่อัปเดตจากสถานะเกมจริง
- เมื่อผ่านเส้นทางที่เลือก ระบบสร้างพงศาวดารตอนจบจากข้อมูลรอบเล่นนั้น
- Save Migration รุ่น 5 รองรับบันทึก v0.9.38 และรุ่นก่อนหน้า
- ปรับ Responsive Layout, ตาราง, แท็บ, ป้ายสถานะ และแถวแรงงานให้ข้อความไม่ล้นหรือซ้อนกัน
- นำข้อมูล “ผู้มาตั้งถิ่นฐานประจำปี” ออกจากหน้าตั้งค่า โดยไม่ลบกลไกประชากรของเกม

## ติดตั้ง

แตก ZIP ลงในโฟลเดอร์แยกจากโครงการเดิม แล้วรัน PowerShell:

```powershell
powershell -ExecutionPolicy Bypass `
  -File ".\apply-v0939-dynasty-endgame.ps1" `
  -ProjectPath "C:\Users\phass\evolution-of-us"
```

ตัวติดตั้งจะสำรองโครงการเดิม ติดตั้งแพ็กเกจ และรันการตรวจระบบทั้งหมด
