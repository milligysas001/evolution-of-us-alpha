# Evolution of Us v0.9.4 — Clarity, Villagers & Settlement Growth

เวอร์ชันนี้ต่อยอดจาก v0.9.3 โดยคงแนวเกม **เริ่มจากศูนย์** แต่ทำให้เล่นง่ายขึ้น อ่านผลลัพธ์ง่ายขึ้น และเพิ่มความสมจริงของคน/ทรัพยากร/ถิ่นฐานมากขึ้น

## จุดเด่น v0.9.4

- เพิ่ม **ปุ่มแนะนำการจัดแรงงาน** ตามสถานการณ์จริงของค่าย
- เพิ่ม **บัญชีทรัพยากรรายเดือน**: คงเหลือ / ผลิต / ใช้ / สุทธิ / หมายเหตุ
- เพิ่ม **Risk Forecast พร้อมเหตุผล** ไม่ใช่แค่แถบตัวเลข
- เพิ่ม **Forecast เดือนหน้า** เพื่อให้วางแผนก่อนฤดูกาลเปลี่ยน
- เพิ่ม **Monthly Result แบบแยกหมวด**: ผลผลิต, บริโภค/สูญเสีย, คนและสุขภาพ, ความก้าวหน้า, เหตุการณ์อื่น
- เพิ่ม **คุณภาพชีวิต**: คุณภาพอาหาร น้ำ และที่พัก
- เพิ่ม **คนสำคัญของค่าย** ให้ชาวบ้านมีตัวตนมากขึ้น
- เพิ่ม **เส้นทางเติบโตของถิ่นฐาน**: ค่ายพักแรม → ชุมชนแรกเริ่ม → หมู่บ้านถาวร → เมืองเล็ก
- เพิ่ม Event Pool รวมเป็น 40+ เหตุการณ์ พร้อม Event Chain ต่อเนื่อง
- ปรับระบบผลกระทบจากตัวเลือกที่เพิ่มความเสี่ยง ให้สะท้อนผ่านสุขภาพ/ความไว้ใจ/ความปลอดภัย/ภัยภายนอกจริง

## ระบบ Realistic ที่มีอยู่

- ชาวบ้านมีชื่อ อายุ สุขภาพ กำลังใจ ความเหนื่อย บาดเจ็บ และเสียชีวิตได้จริง
- สาเหตุเสียชีวิต: อดอาหาร, หนาวจัด, โรค, แผลติดเชื้อ, อุบัติเหตุล่าสัตว์, งานก่อสร้าง, สัตว์ป่า, ชราภาพ
- ฤดูกาลมีผลต่ออาหาร น้ำ ฟืน โรค และสภาพอากาศ
- อาหารเสียถ้าไม่มีคลัง/ถนอมอาหาร
- น้ำและฟืนเป็นทรัพยากรที่ต้องบริหาร
- งานหนักเกินไปเพิ่มความเหนื่อยและอุบัติเหตุ
- คนป่วย/บาดเจ็บรักษาได้ด้วยแรงงานดูแล สมุนไพร และกระท่อมหมอยา
- มีระบบข่าวลือ, Event Chain, Memory, Milestone, ผู้จากไป, เป้าหมายรายระยะ และเส้นทางพัฒนา

## วิธีติดตั้ง

แตก ZIP แล้วรันใน PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v094-clarity-growth.ps1
```

ค่าเริ่มต้นจะติดตั้งไปที่:

```text
C:\Users\phass\evolution-of-us
```

ถ้าโปรเจกต์อยู่คนละที่:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v094-clarity-growth.ps1 -ProjectPath "D:\your-project"
```

หลังขึ้น Ready ให้เปิด:

```text
http://localhost:3000
```

แนะนำเริ่ม Save ใหม่ เพราะ v0.9.4 ใช้ save key ใหม่ `eou-v094-save` เพื่อไม่ให้ข้อมูลเก่าชนกับระบบใหม่
