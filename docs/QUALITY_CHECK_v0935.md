## สิ่งที่ชุดตรวจจับได้

- จำนวนคนเริ่มต้นไม่ตรง 15
- อาหารเริ่มต้นต่ำกว่าค่าปลอดภัย
- วัตถุดิบไม่พอสร้างชุดต้นเกม
- สิ่งก่อสร้างไม่มีต้นทุนหรือมีต้นทุนเป็นศูนย์/ติดลบ
- ทรัพยากรก่อสร้างไม่มีชื่อภาษาไทย
- การมองเห็นสิ่งก่อสร้าง/วิจัยไม่ตรงยุค
- Event ID หรือ Choice ID ซ้ำ
- condition/weight ของ Event โยน exception
- Choice ทำให้ทรัพยากรหรือค่าสถานะเป็น `NaN`/ติดลบ
- สัตว์ออกลูกทั้งที่ไม่มีคู่
- ระบบเชือดหมูไม่ตรงข้อมูลกลาง

## ขอบเขต

ชุดตรวจอัตโนมัติยืนยัน logic, data และ build ไม่ได้แทนการทดสอบการอ่านง่ายบนจอจริงทุกความละเอียด หลังติดตั้งควรเปิด `/game` และตรวจหน้า “คน”, “ทรัพยากร”, “ก่อสร้าง”, “วิจัย” และ “สัตว์เลี้ยง” อีกครั้งบนขนาดจอที่ใช้งานจริง
