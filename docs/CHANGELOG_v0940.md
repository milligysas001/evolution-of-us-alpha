# Evolution of Us v0.9.40 — Narrative & Interface Polish

วันที่: 14 กรกฎาคม 2026

## รายงานจบเดือน

- นำคำเตือน “ยังไม่ได้เลือกการกระทำผู้นำ” และ “ยังไม่ได้ตอบเหตุการณ์” ออกจากรายงานจบเดือน
- คำเตือนสองรายการนี้ยังแสดงในหน้าตัดสินใจ และปุ่มจบเดือนยังถูกล็อกจนกว่าจะเลือกครบ
- รายงานจบเดือนจึงเหลือเฉพาะผลที่เกิดขึ้นจริงและความเสี่ยงที่ต้องแก้ในเดือนถัดไป

## จัดแรงงาน

- นำข้อความ “คำนวณผลจริงตามความถนัด” ออกจากแถวตัวละคร
- แถวคนยังแสดงสุขภาพ ความล้า งานปัจจุบัน และอัตราพักฟื้นครบ
- แก้ช่องเลือกงานแบบกลุ่มที่เคยถูกบีบจนข้อความแสดงไม่เต็ม
- ปุ่มและช่องเลือกปรับความสูงตามข้อความ รองรับ Desktop, Tablet และ Mobile

## แถบสถานะด้านบน

- แยกข้อมูลเป็นสถานะหลักและข้อมูลประกอบสองระดับ
- สถานะหลัก: ปี/เดือน ระยะเมือง ประชากร แรงงาน วิกฤต และสถานะบันทึก
- ข้อมูลประกอบ: อากาศ ระดับความยาก พื้นที่สำรวจ ผลผลิต ตระกูล ภัยภายนอก คลังเมือง รุ่นเกม และมุมมอง
- ลดความแน่นของข้อความและป้องกันข้อมูลชนปุ่มแจ้งเตือน/ตั้งค่า

## พงศาวดารตอนจบ

- เพิ่มพงศาวดารฉบับเต็มในหน้าจบเกม
- อ่านสมาชิกตระกูล ประวัติผู้นำ เส้นทางชัยชนะ ความทรงจำ ผู้จากไป และลำดับเหตุการณ์ได้จากหน้าเดียว
- ลำดับเหตุการณ์จัดจากต้นเกมไปจบเกมและแบ่งตามปี
- เพิ่มเพดานเก็บบันทึกเป็น 6,000 รายการ และความทรงจำ 1,000 รายการ เพื่อรองรับเกมยาว

## ทางเลือกเหตุการณ์

- นำทางเลือกกลางที่ซ้ำกันออกจาก Event Generator ได้แก่:
  - รับมืออย่างระมัดระวัง
  - ใช้แรงงานแก้ปัญหาทันที
  - ให้ชุมชนร่วมตัดสินใจ
- ทางเลือกใหม่เปลี่ยนตามบริบทของอาหาร น้ำ ครอบครัว ผู้บาดเจ็บ สัตว์ ก่อสร้าง สำรวจ และเหตุการณ์ทั่วไป
- แสดงผลที่จะเกิดขึ้นเป็นภาษาไทยตรงไปตรงมา เช่น อาหาร -4, กำลังใจ +2 หรือความเสี่ยงโรคลด 3
- เหตุการณ์ “ช่างที่มือแตก: มิตรภาพที่เกิดจากงานหนัก” ใช้ทางเลือกที่สอดคล้องกับบาดแผล งาน และการช่วยเหลือของชุมชน
