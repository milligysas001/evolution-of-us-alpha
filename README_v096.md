# Evolution of Us v0.9.6 — Game Over & Collapse System

เวอร์ชันนี้ต่อยอดจาก v0.9.5 โดยเพิ่มระบบแพ้/ล่มสลายของค่ายให้สมจริงขึ้น พร้อมแก้บัค React duplicate key ที่เกิดจาก tag ภาษาไทยซ้ำ เช่น `สุขาภิบาล`

## จุดที่อัปเดต

- เพิ่ม Game Over เมื่อประชากรเหลือ 0
- เพิ่มเงื่อนไขแพ้หลายแบบ:
  - ประชากรสูญสิ้น
  - ความอดอยากต่อเนื่อง
  - ผู้นำถูกขับออกเพราะความไว้ใจและความสามัคคีพัง
  - ค่ายแตกจากภัยภายนอกเมื่อภัยสูงและความปลอดภัยต่ำ
  - ไร้แรงงานและหมดทางรอด
- เพิ่มระดับวิกฤตของค่าย: มั่นคง / น่ากังวล / วิกฤต / ใกล้ล่มสลาย
- เพิ่มหน้า Game Over แบบพงศาวดาร พร้อมสรุปว่าค่ายอยู่รอดกี่ปี ประชากรสุดท้าย ผู้จากไป สิ่งก่อสร้าง และสาเหตุการล่มสลาย
- เพิ่มปุ่มเริ่มเกมใหม่จากชื่อเดิม และปุ่มกลับหน้าแรก
- ปรับบันทึกผู้จากไปให้แสดงแบบย่อ ไม่ไหลยาวเกินกรอบ และดูทั้งหมดผ่าน details ได้
- แก้บัค duplicate key ใน tag/summary/forecast/death list
- เปลี่ยน save key เป็น `eou-v096-save` เพื่อไม่ชนกับเซฟเก่า

## วิธีติดตั้ง

แตก ZIP แล้วรันใน PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File .\apply-v096-gameover-collapse.ps1
```

จากนั้นเปิด:

```text
http://localhost:3000
```

แนะนำให้เริ่มเกมใหม่เพื่อให้ระบบล่มสลายและ save key ใหม่ทำงานเต็มครับ
